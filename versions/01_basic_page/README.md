# 01 Basic Page

Базовая страница загрузки данных для первого среза проекта. Приложение принимает один файл или вставленный текст, проверяет тип, размер и минимальную читаемость, после чего показывает нормализованный результат.

## Запуск

```bash
npm install
npm run dev
```

Страница открывается на `/`.

## Проверки

```bash
npm run typecheck
npm run build
```

Playwright-тесты из корня репозитория:

```powershell
.\tests\01_basic_page\run_data_loader_tests.ps1 -BaseUrl http://localhost:3000 -BasicPagePath /
```

## Ограничения

- Файлы не отправляются на backend.
- Полный парсинг XLS/XLSX/ODS/DOC/DOCX не выполняется в этом срезе.
- Лимит файла: 10 МБ.
- Лимит вставленного текста: 100 000 символов.
